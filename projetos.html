Agência OBelico

Desenvolvi um projeto para a Agência OBelico, especializada em serviços de despachante de armas, com foco em uma interface moderna, intuitiva e otimizada para SEO.

O site foi projetado para oferecer uma experiência de navegação fluida e profissional, destacando os serviços da agência de forma clara e objetiva.

Implementei práticas avançadas de SEO, como estruturação semântica do HTML, uso estratégico de meta tags e otimização de performance, garantindo maior visibilidade nos motores de busca.

Utilizei tecnologias como HTML, CSS e JavaScript para criar um design responsivo e funcional, compatível com diferentes dispositivos e navegadores.
